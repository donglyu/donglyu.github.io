整理下Swift与Objective-C的一些不同

2016-08-23
  1. ChangeLog

在OC中对代码进行逻辑组织用的是#pragma mark - , 生成分割线
而在swift中有了一些新的语法
如: //MARK: //FIXME TODO:

//MARK: - 生成分割线

//MARK: 说明

  • Swift中对类进行扩展
    OC中采用的是分类, 而Swift中没有分类, 相对应的只有扩展(Extensions)
    以UIView为例 , 新建UIView+Ext.swift
import Foundation
import UIKit


extension UIView {
    public var x: CGFloat{
        get{
            return self.frame.origin.x
        }
        set{
            var r = self.frame
            r.origin.x = newValue
            self.frame = r
        }
    }
    // ....
}
